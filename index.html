jQuery.messagebox

jQuery.messagebox is a jQuery UI plugin that shows simple and customizable modal dialogs.

Demo



Requirements

  • jQuery
  • jQuery UI

How to use it

Include javascript code and stylesheet

    <script src="js/jquery.messagebox.js"></script>
    <link rel="stylesheet" href="css/jquery.messagebox.css" />

Examples

Simple information message

    $.infoMessage("This is an information message");

    $.infoMessage("This is an information message with success callback", 
            function() {
                    alert("success")
            }
    );
You can show warning, error and confirm messages with: 
    
	$.warningMessage(...); 
	$.errorMessage(...); 
	$.confirmMessage(...);

Customizing colors

You can customize colors by editing jquery.messagebox.css

/* ERROR DIALOG */
.messagebox-error { 
    border: 1px solid #CD0A0A; /* BORDER COLOR */
    background: #FEF1EC; /* BACKGROUND COLOR */
    color: #CD0A0A; /* TEXT COLOR */
}
